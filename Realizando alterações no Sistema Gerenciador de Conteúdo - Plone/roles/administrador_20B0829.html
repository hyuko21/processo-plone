Role: Administrador
Pessoa com permissões ilimitadas. Tem acesso a todo o sistema, podendo realizar alterações em todos os aspectos mas, por padrão, tem a responsabilidade de definir o papel de outros usuários no sistema.
Role Sets: Administrador
Relationships
Additionally Performs
  • Entregar modificações
  • Realizar modificações
Main Description

Gerente de permissões de usuários e alterações instrísicas ao sistema CMS.

Key Considerations

Define os papeis de outros usuários, permitindo o acesso destes a determinadas funcionalidades do CMS.
