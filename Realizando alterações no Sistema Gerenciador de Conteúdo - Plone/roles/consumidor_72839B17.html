Role: Consumidor
Consumidor de conteúdo do sistema CMS. É uma especificação do papel de Usuário.
Role Sets: Consumidor
Relationships
Primary Performs
  • Consumir conteúdo
  • Identificar problema
  • Requisitar alteração
Modifies
  • Conteúdo consumido
  • Problema
  • Requisição de mudança
    Main Description

    O consumidor tem interesse em encontrar informações relevantes para si, embora ainda, este possa colaborar com outros usuários, para aprimorar o conteúdo do sistema.

    Key Considerations
    A este tipo de usuário, além de consumir o conteúdo publicado no sistema, são dadas as responsabilidades de análise desse conteúdo, para que o mesmo possa ser ajustado, caso tenham sido encontradas falhas de conteúdo.
